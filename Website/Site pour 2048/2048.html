Jouez au 2048

Principe du jeu:




Le but de ce jeu de grille est d'atteindre par une succession de mouvement vers la droite, le bas, le haut ou la gauche le nombre 2048 dans la grille.



Pour qu'une opération de mouvement soit réalisable, il faut que deux nombres accolés aient la même valeur pour qu'ils se somment.Par aileurs on part d'une grille remplie partiellement de 2 et/ou de 4.

Calcul des scores:

Pour calculer votre score final, l'opération est assez simple. Il suffit de sommer la valeur de toutes les cases qui correspondra à votre nombre de points. Le score n'est calculé que si vous atteignez 2048, sinon le score vaut 0 et vous ne pourrez pas être classé.
Dans le premier cas, le joueur atteint l'objectif c'est à dire au moins une case est suprieur ou égale à 2048. Dans ce cas son score est calculé selon le modèle présenté précedemment et ce dernier est de: 4112.
Dans le second cas, le joueur n'atteint pas l'objectif et son score est de 0.

Scores:

Les scores seront actualisés à chaque fois qu'un joueur ajoute son pseudo et finit une partie(qui peut s'achever sur une défaite ou sur une victoire).Le jeu biensûr continue jusqu'à ce que l'on ne puisse plus jouer.
